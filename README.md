<h1> 
Pixabay API
</h1>

<h2>
Intro
</h2>
Pixabay API is a personal project developed with React. It allows you to get images using pixabay API.


<h2>
Getting Started
</h2>

```
git clone https://github.com/BauTancredi/pixabay-api.git
cd pixabay-api
npm install
npm start
```
